## 🕯️ Featured Project — The Hollow Escape

> *A first and third person survival horror game set inside an abandoned asylum. Five minutes. Locked rooms. No second chances.*

**The Hollow Escape** is a fully shipped Unity game built entirely from the ground up. Every system — from enemy AI to camera physics to audio architecture — was designed, implemented, and debugged independently.

### ⚙️ Technical Highlights

| System | Details |
|---|---|
| **Enemy AI** | Universal AI script handling patrol, vision detection, chase, attack, hit reaction and death — configurable per enemy type via Inspector |
| **Camera System** | Dual-mode TPP / FPS camera with SphereCast wall collision, seamless mode switching, and crosshair activation logic |
| **Room Architecture** | Per-room barrier controllers, entry triggers, and enemy lifecycle management operating as independent units |
| **Audio System** | Singleton SoundManager with per-room dynamic music, background fading, loop interval control, and dedicated 2D SFX routing |
| **Combat** | Raycast shooting, muzzle flash particles, dynamic muzzle light, ammo pickup, reload pipeline, and alternating punch combos |
| **Hazards** | Poison gas with runtime transparent material generation, time remove buttons, health restore system, fall damage |
| **UI & Pause** | Countdown timer, health bar, ammo display, pause menu with full cursor state management across scene transitions |
