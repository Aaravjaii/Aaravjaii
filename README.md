### 🚀 Featured Project

#### 🕯️ The Hollow Escape — Survival Horror Game
> Built in Unity 6.3 LTS · C# · DX11 · Windows / Mac / Linux

A first and third person survival horror game set inside an abandoned asylum. Players navigate locked rooms, survive environmental hazards, and escape before a countdown timer hits zero.

**Key systems built from scratch:**
- Universal enemy AI — patrol, detection, chase, attack, and death with full animation pipeline
- Dual-mode camera system with SphereCast wall collision in both TPP and FPS modes
- Per-room barrier and entry trigger architecture with independent enemy lifecycle management
- Singleton SoundManager with dynamic per-room music, background fading, and 2D SFX routing
- Raycast shooting system with muzzle flash, hit effects, ammo management, and reload pipeline
- Poison gas controller, interactive physical buttons, countdown timer, and full pause system
